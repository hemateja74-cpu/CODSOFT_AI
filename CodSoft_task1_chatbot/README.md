# CODSOFT Task 1 – Rule-Based Chatbot 🤖

## 📌 Overview
This is a simple **rule-based chatbot** built using Python.  
It uses **regular expressions (regex)** to match user input against predefined rules and respond accordingly.  
The chatbot runs directly in the terminal and demonstrates basic AI concepts without requiring machine learning.

---

## ✨ Features
- Responds to greetings (`hi`, `hello`, `hey`)
- Answers basic queries like *how are you* or *who are you*
- Provides help information
- Handles exit commands (`bye`, `quit`, `exit`)
- Regex pattern matching for flexible input handling
- Lightweight and beginner-friendly

---

## 🛠 Tech Stack
- **Language:** Python 3.8+
- **Libraries:** `re` (Python standard library)
