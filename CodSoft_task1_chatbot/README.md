# CODSOFT Task 1 – Rule-Based Chatbot

## 📌 Overview
A simple **rule-based chatbot** built using Python.  
It responds to user inputs based on predefined rules using `if-else` statements.

## 🚀 Features
- Greets the user
- Answers basic queries
- Ends conversation when user types "bye" or "exit"

## ▶️ How to Run
```bash
python chatbot.py
