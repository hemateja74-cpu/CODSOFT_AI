Task 3 - Recommendation System

Overview
This project implements a simple recommendation system using content-based filtering.
It suggests movies based on their similarity in genre/keywords.

Requirements
Install dependencies:
pip install -r requirements.txt

How to Run
python recommendation.py

Example Output
Recommendations for Titanic: ['The Notebook', 'Inception']
Recommendations for Inception: ['Interstellar', 'Avatar']

Files
recommendation.py  → main script
requirements.txt   → dependencies
README.md          → documentation
