## 📌 Overview
This repository contains three AI projects developed as part of the **CODSOFT Internship**.  
Each task is organized in its own folder with code, requirements, and documentation.

---

## 📂 Tasks
### 🔹 Task 1: Rule-Based Chatbot
- Implements a simple chatbot in Python.
- Responds to user queries based on predefined rules and keywords.
- Demonstrates basic natural language processing concepts.

### 🔹 Task 2: Tic-Tac-Toe AI
- Classic Tic-Tac-Toe game with an AI opponent.
- Uses the **Minimax algorithm** to play optimally.
- Provides an interactive command-line gameplay experience.

### 🔹 Task 3: Recommendation System
- Content-based movie recommendation system.
- Suggests movies based on similarity of features.
- Built using **scikit-learn** and Python libraries.
